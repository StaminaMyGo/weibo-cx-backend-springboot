# 微博系统 RESTful API 接口文档（v1 版本）

## 1. 基础信息

- **接口前缀**：所有接口统一前缀为 `/api/v1`
- **数据格式**：除图片上传接口外，请求与响应均采用 `application/json` 格式
- **鉴权规则**：需登录后访问的接口，需在请求头携带 `Authorization: Bearer {jwt_token}`（token 由登录接口返回）
- **数据库关联**：接口字段与数据库表（users、weibos、comments、attentions）字段一一对应，无额外冗余字段

## 2. 统一响应结构

```json
{
  "code": 200,      // 状态码（详情见下方说明）
  "msg": "操作成功", // 提示信息（失败时返回错误原因）
  "data": {}        // 业务数据（成功时返回，失败时可为 null）
}
```

## 3. 状态码说明

| 状态码 | 含义 | 常见场景 |
| :--- | :--- | :--- |
| 200 | 操作成功 | 登录、注册、发布微博、关注等正常操作 |
| 400 | 参数错误 | 必填参数缺失、参数格式错误 |
| 401 | 未授权 | 未携带 token、token 过期、token 无效 |
| 403 | 无权限 | 删除他人微博、评论，关注已关注用户等非法操作 |
| 404 | 资源不存在 | 访问不存在的微博、用户、评论 |
| 500 | 服务器异常 | 数据库操作失败、接口内部报错 |

## 4. 用户模块（users）

### 4.1 用户注册
- **URL**: `POST /api/v1/users/register`
- **鉴权**: 无需鉴权
- **说明**: 插入数据到 users 表，生成 id（自增）

**请求体参数**:
| 参数名 | 类型 | 必填 | 长度限制 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| loginname | string | 是 | 1-20 字符 | 用户登录账号，唯一不可重复 |
| loginpwd | string | 是 | 1-20 字符 | 用户登录密码 |
| nickname | string | 是 | 1-20 字符 | 用户昵称，可重复 |

**请求示例**:
```json
{
  "loginname": "zhangsan",
  "loginpwd": "123456",
  "nickname": "张三"
}
```

**响应示例**:
```json
{
  "code": 200,
  "msg": "注册成功",
  "data": {
    "id": 8,
    "loginname": "zhangsan",
    "nickname": "张三"
  }
}
```

### 4.2 用户登录（获取 JWT 令牌）
- **URL**: `POST /api/v1/users/login`
- **鉴权**: 无需鉴权
- **说明**: 查询 users 表，匹配 loginname 和 loginpwd

**请求体参数**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| loginname | string | 是 | 用户登录账号 |
| loginpwd | string | 是 | 用户登录密码 |

**请求示例**:
```json
{
  "loginname": "zhangsan",
  "loginpwd": "123456"
}
```

**响应示例**:
```json
{
  "code": 200,
  "msg": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "id": 1,
    "nickname": "张三",
    "photo": "7.jpg",
    "score": 240103100,
    "attionCount": 108
  }
}
```

### 4.3 获取当前登录用户信息
- **URL**: `GET /api/v1/users/current`
- **鉴权**: 需鉴权
- **说明**: 根据 token 解析的 id，查询 users 表完整信息

**响应示例**:
```json
{
  "code": 200,
  "msg": "查询成功",
  "data": {
    "id": 1,
    "nickname": "张三",
    "loginname": "zhangsan",
    "photo": "7.jpg",
    "score": 240103100,
    "attionCount": 108
  }
}
```

### 4.4 根据 ID 获取用户信息
- **URL**: `GET /api/v1/users/{userId}`
- **鉴权**: 无需鉴权
- **说明**: 根据路径参数 userId，查询 users 表对应记录

**响应示例**:
```json
{
  "code": 200,
  "msg": "查询成功",
  "data": {
    "id": 1,
    "nickname": "张三",
    "photo": "7.jpg",
    "score": 240103100,
    "attionCount": 108
  }

```

### 4.5 关注用户
- **URL**: `POST /api/v1/users/{userId}/follow`
- **鉴权**: 需鉴权
- **说明**: 当前登录用户关注指定用户，写入 attentions 表，同时将目标用户的 attionCount +1

**路径参数**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| userId | int | 是 | 被关注用户的 ID |

**响应示例**:
```json
{
  "code": 200,
  "msg": "关注成功",
  "data": null
}
```

### 4.6 取消关注
- **URL**: `DELETE /api/v1/users/{userId}/follow`
- **鉴权**: 需鉴权
- **说明**: 当前登录用户取消关注指定用户，删除 attentions 表对应记录，同时将目标用户的 attionCount -1

**路径参数**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| userId | int | 是 | 被取消关注用户的 ID |

**响应示例**:
```json
{
  "code": 200,
  "msg": "取消关注成功",
  "data": null
}
```

### 4.7 获取用户的粉丝列表

- **说明**: 查询关注了该用户的所有用户列表（att_marstid = userId）

```json
}
```

### 4.8 获取用户的关注列表

## 5. 微博模块（weibos）

### 5.1 发布微博
- **URL**: `POST /api/v1/weibos`
- **鉴权**: 需鉴权
- **说明**: 插入数据到 weibos 表，userid 为当前登录用户 ID

**请求体参数**:
| 参数名 | 类型 | 必填 | 长度限制 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| title | string | 是 | 1-200 字符 | 微博标题 |
| content | string | 是 | 1-2000 字符 | 微博内容（可包含 HTML 标签） |
| img | string | 否 | 1-100 字符 | 微博图片路径，无图片则为 null |

**请求示例**:
```json
{
  "title": "一个博物馆就是一所大学校",
  "content": "<p>走进博物馆，了解历史...</p>",
  "img": "c3.jpg"
}
```

**响应示例**:
```json
{
  "code": 200,
  "msg": "发布成功",
  "data": {
    "id": 70,
    "userid": 1,
    "title": "一个博物馆就是一所大学校",
    "createtime": "2024-05-03 14:30:00"
  }
}
```

### 5.2 微博列表（分页 + 搜索）
- **URL**: `GET /api/v1/weibos`
- **鉴权**: 无需鉴权
- **说明**: 查询 weibos 表，支持按 title、content 模糊搜索，分页返回

**请求参数**:
| 参数名 | 类型 | 必填 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| pageNum | int | 是 | 1 | 页码 |
| pageSize | int | 是 | 10 | 每页显示条数 |
| keyword | string | 否 | null | 搜索关键词，模糊匹配微博标题和内容 |

**响应示例**:
```json
{
  "code": 200,
  "msg": "查询成功",
  "data": {
    "total": 1,
    "pages": 1,
    "pageNum": 1,
    "pageSize": 5,
    "list": [
      {
        "id": 59,
        "userid": 1,
        "title": "一个博物馆就是一所大学校",
        "content": "<p>...</p>",
        "createtime": "2022-05-18 23:08:42",
        "readcount": 0,
        "img": "c3.jpg",
        "user_info": {
          "id": 1,
          "nickname": "张三",
          "photo": "7.jpg"
        }
      }
    ]
  }
}
```

### 5.3 微博详情
- **URL**: `GET /api/v1/weibos/{wbId}`
- **鉴权**: 无需鉴权
- **说明**: 根据 wbId 查询 weibos 表，同时关联 users 表获取发布者信息，访问时自动将 readcount +1

**响应示例**:
```json
{
  "code": 200,
  "msg": "查询成功",
  "data": {
    "id": 59,
    "userid": 1,
    "title": "一个博物馆就是一所大学校",
    "content": "<p>...</p>",
    "createtime": "2022-05-18 23:08:42",
    "readcount": 1,
    "img": "c3.jpg",
    "user_info": {
      "id": 1,
      "nickname": "张三",
      "photo": "7.jpg",
      "attionCount": 108
    }
  }
}
```

### 5.4 删除自己的微博
- **URL**: `DELETE /api/v1/weibos/{wbId}`
- **鉴权**: 需鉴权
- **说明**: 根据 wbId 删除 weibos 表对应记录，需验证 userid 与当前登录用户 ID 一致

**响应示例**:
```json
{
  "code": 200,
  "msg": "删除成功",
  "data": null
}
```

## 6. 图片上传模块

### 6.1 上传图片
- **URL**: `POST /api/v1/upload/images`
- **鉴权**: 需鉴权
- **格式**: multipart/form-data
- **说明**: 图片路径用于填充 weibos 表的 img 字段

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| file | file | 是 | 图片文件（支持 jpg、png，大小不超过 5MB） |

**响应示例**:
```json
{
  "code": 200,
  "msg": "上传成功",
  "data": {
    "imgUrl": "c3.jpg"
  }
}
```

## 7. 评论模块（comments）

### 7.1 发表评论
- **URL**: `POST /api/v1/weibos/{wbId}/comments`
- **鉴权**: 需鉴权
- **说明**: 插入数据到 comments 表，weiboid 为路径参数 wbId，userid 为当前登录用户 ID

**请求参数**:
| 参数名 | 类型 | 必填 | 位置 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| wbId | int | 是 | 路径参数 | 目标微博 ID |
| content | string | 是 | 请求体 | 评论内容（可包含 HTML 标签、表情） |

**请求示例**:
```json
{
  "content": "<p>很好</p>"
}
```

**响应示例**:
```json
{
  "code": 200,
  "msg": "评论成功",
  "data": {
    "id": 5,
    "weiboid": 59,
    "userid": 1,
    "content": "<p>很好</p>"
  }
}
```

### 7.2 获取微博的评论列表
- **URL**: `GET /api/v1/weibos/{wbId}/comments`
- **鉴权**: 无需鉴权
- **说明**: 查询指定微博下的所有评论，按评论 ID 降序排列，关联返回评论者信息

**路径参数**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| wbId | int | 是 | 目标微博 ID |

**响应示例**:
```json
{
  "code": 200,
  "msg": "查询成功",
  "data": {
    "total": 2,
    "list": [
      {
        "id": 5,
        "weiboid": 59,
        "userid": 1,
        "content": "<p>很好</p>",
        "user_info": {
          "id": 1,
          "nickname": "张三",
          "photo": "7.jpg"
        }
      }
    ]
  }
}
```

### 7.3 删除评论
- **URL**: `DELETE /api/v1/comments/{commentId}`
- **鉴权**: 需鉴权
- **说明**: 删除指定评论，仅允许评论作者或微博作者删除

**路径参数**:
| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| commentId | int | 是 | 评论 ID |

**响应示例**:
```json
{
  "code": 200,
  "msg": "删除成功",
  "data": null
}
```
